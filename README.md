# InventoryQR Gymnasium 8

Мобильное приложение для учёта оборудования в МОУ «Тверская гимназия №8» с использованием QR-кодов.

## О проекте
- **Цель**: Автоматизация инвентарного учёта, инвентаризации, контроля перемещений и отчётности.
- **Технологии**: Android Studio, Java, ZXing (QR), SQLite (в разработке).
- **Практика**: Производственная практика, 2025-2026, должность — помощник системного администратора.

## Функции приложения
1. Авторизация
2. Главный экран (дашборд)
3. Сканирование QR-кода → карточка оборудования
4. Добавление оборудования + генерация QR
5. Список оборудования с поиском
6. Отчёты (общее количество, по статусам, по кабинетам)
